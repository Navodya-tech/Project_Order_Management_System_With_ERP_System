# Order Management System with ERP

## 📌 Project Description
This is my **Order Management System with ERP features**, a full‑featured desktop application built using **Java Swing** and **MySQL**. It helps restaurants and small businesses manage their core operations — from customer ordering and kitchen workflow to inventory, HR, payroll, and administration.

---

## 🛠️ What It Does

This system is designed to streamline restaurant and business operations, including:

- Customer **ordering and dining interface**
- Kitchen order management and preparation tracking
- Inventory tracking and stock control
- Human Resources: attendance, leave management
- Finance & payroll management
- Role‑based dashboards for real‑time insights

Both **customers** and **staff** can interact with the system for their respective workflows.

---

## 🚀 Key Features

### 👤 User Roles
- Super Admin  
- HR Manager  
- Finance Manager  
- Inventory Manager  
- Kitchen Staff  
- Customer (Dining Interface)

### 🧩 System Features
- ✅ Role‑based login access  
- 🧾 Invoice generation & payment handling  
- 🍽️ Dining and kitchen order workflows  
- 📦 Inventory control (GRNs, purchases, stock updates)  
- 👥 HR module (attendance, leave, user creation)  
- 💰 Finance module (payslips, salary advances, expenses)  
- 📊 Dashboards with KPIs, graphs, and summaries  
- 🔍 Search, filter, export functions  
- 🛡️ Data validation and error handling

## 📥 How to Run

1. **Clone the repository**
```bash
git clone https://github.com/Navodya-tech/Project_Order_Management_System_With_ERP_System/edit/main/README.md.git
```

2. Import the database
- Open HeidiSQL or MySQL Workbench
- Import /database/order_management_system.sql
- Make sure “Data” is checked to include all table rows

3. Configure the backend
- Open the project in your IDE (Netbeans)
- Update database connection settings in MySql.java file

4. Run the application
